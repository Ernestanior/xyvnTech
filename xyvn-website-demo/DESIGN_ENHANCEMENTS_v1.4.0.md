## 🎨 全局样式升级

### 1. 深色主题

#### 背景渐变
```css
background: linear-gradient(
  to bottom right,
  #111827,  /* gray-900 */
  #111827,  /* gray-900 */
  #1f2937   /* gray-800 */
);
```

#### 优势
- ✅ 减少眼睛疲劳
- ✅ 突出彩色元素
- ✅ 现代化视觉
- ✅ 节省电量（OLED 屏幕）

---

### 2. 自定义动画

#### 浮动动画
```css
@keyframes float {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-20px); }
}
```

#### 发光动画
```css
@keyframes glow {
  from { box-shadow: 0 0 20px rgba(59, 130, 246, 0.5); }
  to { box-shadow: 0 0 30px rgba(59, 130, 246, 0.8); }
}
```

#### 渐变动画
```css
@keyframes gradient {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}
```

## 💫 动画原则

### 1. 性能优先
- 使用 transform 和 opacity
- 避免 layout 和 paint
- 启用 GPU 加速

### 2. 时长控制
```
快速：0.15s - 0.3s  (按钮、链接)
标准：0.3s - 0.6s   (卡片、模态框)
慢速：0.6s - 1.2s   (页面过渡)
```

### 3. 缓动函数
```
ease-out    - 进入动画
ease-in     - 退出动画
ease-in-out - 双向动画
spring      - 弹性动画
```
