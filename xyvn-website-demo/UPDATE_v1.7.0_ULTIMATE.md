## 🎨 设计理念

### 极致交互
1. **磁吸效果** - 按钮跟随鼠标
2. **光晕跟随** - 卡片光晕跟随
3. **视差滚动** - 深度感体验
4. **弹簧动画** - 物理真实感

### 视觉层次
1. **动态背景** - 网格、光晕、渐变
2. **浮动元素** - 装饰图标浮动
3. **渐变色彩** - 统一的色彩系统
4. **微交互** - 细节动画反馈

### 性能优化
1. **GPU 加速** - transform 动画
2. **视口触发** - 只在可见时动画
3. **只播放一次** - 避免重复计算
4. **节流优化** - 鼠标事件节流
