## 📝 如何添加新案例

在 `src/data/portfolioData.ts` 文件中添加：

```typescript
{
  id: 9,
  title: '新项目名称',
  category: 'website', // website | app | ecommerce | enterprise
  client: '客户名称',
  industry: '所属行业',
  description: '项目详细描述...',
  image: 'https://images.unsplash.com/photo-xxx?w=1200&q=80',
  tags: ['技术1', '技术2', '技术3'],
  metrics: [
    { icon: 'TrendingUp', label: '指标名称', value: '数值' },
    { icon: 'Users', label: '指标名称', value: '数值' },
    { icon: 'Award', label: '指标名称', value: '数值' },
  ],
  highlights: [
    '核心亮点1',
    '核心亮点2',
    '核心亮点3',
    '核心亮点4',
  ],
  year: '2025',
  duration: '开发周期',
}
```
