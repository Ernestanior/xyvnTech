## 🎯 3 步启动

### 步骤 1：安装依赖

```bash
cd xyvn-website-demo
npm install
```

这将安装所有必需的包：
- next (Next.js 14)
- react & react-dom
- typescript
- tailwindcss

### 步骤 2：启动开发服务器

```bash
npm run dev
```

访问：http://localhost:3000

## 🎨 自定义

### 修改颜色

编辑 `tailwind.config.ts`:

```typescript
colors: {
  apple: {
    blue: '#0071E3',  // 改成你的主色
  },
}
```

### 修改文案

编辑对应的组件文件：
- Hero 标题：`src/components/sections/HeroSection.tsx`
- 服务内容：`src/components/sections/ServicesSection.tsx`
- 案例内容：`src/components/sections/PortfolioSection.tsx`

### 替换图片

案例图片使用的是 Unsplash 的图片，你可以：
1. 替换 URL 为你自己的图片
2. 或者保持使用 Unsplash 的图片

## 🐛 常见问题

### Q: npm install 失败？
```bash
# 清除缓存
npm cache clean --force
# 重新安装
npm install
```

### Q: 端口被占用？
```bash
# 使用其他端口
npm run dev -- -p 3001
```

### Q: 图片加载失败？
在 `next.config.js` 中已配置 Unsplash 域名，应该可以正常加载。

### Q: 样式不生效？
```bash
# 清除缓存
rm -rf .next
npm run dev
```
