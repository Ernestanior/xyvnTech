# Public Assets

## Required Images

### og-image.png
- **Size**: 1200x630 pixels
- **Purpose**: Open Graph image for social media sharing
- **Format**: PNG
- **Description**: This image will be displayed when your website is shared on social media platforms like Facebook, Twitter, LinkedIn, etc.

**Action Required**: Please create and add an `og-image.png` file to this directory with your branding and key message.

### Recommended Content for OG Image:
- Company logo (XYVN Tech)
- Tagline: "专业的软件开发服务" / "Professional Software Development Services"
- Visual elements that represent your services (web, app, mini-program icons)
- Clean, professional design with good contrast
- Text should be readable at smaller sizes
