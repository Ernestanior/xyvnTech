## 🚀 性能优化

### 动画性能

```
优先使用 GPU 加速属性：
✓ transform
✓ opacity
✓ filter

避免触发重排：
✗ width/height
✗ top/left
✗ margin/padding
```

### 资源加载

```
优先级：
1. 关键 CSS (内联)
2. 首屏图片 (preload)
3. 字体文件 (preload)
4. 其他资源 (懒加载)

图片策略：
• WebP/AVIF 格式
• 响应式 srcset
• 模糊占位符
• 懒加载 (Intersection Observer)
```
