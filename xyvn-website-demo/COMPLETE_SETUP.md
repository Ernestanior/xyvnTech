# 完整安装指南

## 第一步：进入项目目录

```bash
cd xyvn-website-demo
```

## 第二步：安装依赖

```bash
npm install
```

我已经为你创建了以下文件：
- ✅ package.json
- ✅ tsconfig.json

- ✅ postcss.config.js
- ✅ src/app/layout.tsx
- ✅ src/app/page.tsx

- ✅ src/lib/utils.ts
- ✅ src/components/Navbar.tsx

## 第四步：运行项目

```bash
npm run dev
```

然后访问 http://localhost:3000
