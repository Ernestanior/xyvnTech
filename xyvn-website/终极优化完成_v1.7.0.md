### 🎨 5个高级交互组件

#### 1. MagneticButton - 磁吸按钮
```tsx
<MagneticButton className="...">
  开始项目
</MagneticButton>
```

**特性**:
- 鼠标靠近时跟随移动
- 物理弹簧动画
- 可配置吸引强度
- 平滑归位效果

#### 2. GlowCard - 光晕卡片
```tsx
<GlowCard className="...">
  {/* 内容 */}
</GlowCard>
```

**特性**:
- 光晕跟随鼠标位置
- 600px 径向渐变
- 边框光效
- 平滑过渡

#### 3. NumberTicker - 数字滚动
```tsx
<NumberTicker 
  value={500} 
  suffix="+" 
  duration={2000}
/>
```

**特性**:
- 缓动动画
- 视口触发
- 只播放一次
- 可配置时长

#### 4. TextReveal - 文字揭示
```tsx
<TextReveal delay={0.3}>
  重新定义数字化体验
</TextReveal>
```

**特性**:
- 逐字显示
- 弹簧动画
- 可配置延迟
- 视口触发

#### 5. ParallaxSection - 视差区块
```tsx
<ParallaxSection speed={0.5}>
  {/* 内容 */}
</ParallaxSection>
```

**特性**:
- 视差滚动
- 可配置速度
- transform 优化
- 平滑过渡
