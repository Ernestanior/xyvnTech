## 🚀 如何使用

### 方法一：快速启动（推荐）

```bash
# 1. 进入项目目录
cd xyvn-website-demo

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 打开浏览器
# 访问 http://localhost:3000
```

### 方法二：手动创建

如果需要手动创建所有文件，请按照以下步骤：

1. 查看 `ALL_COMPONENTS_CODE.md`
2. 复制所有代码到对应文件
3. 运行 `npm install`
4. 运行 `npm run dev`

## 🐛 故障排查

### 问题：npm install 失败
**解决**：
```bash
# 清除缓存
npm cache clean --force
# 重新安装
npm install
```

### 问题：粒子背景不显示
**解决**：检查浏览器控制台，确保没有 JavaScript 错误

### 问题：样式不生效
**解决**：
```bash
# 清除 Next.js 缓存
rm -rf .next
# 重新启动
npm run dev
```

### 问题：TypeScript 报错
**解决**：确保所有文件都已创建，参考 `ALL_COMPONENTS_CODE.md`
