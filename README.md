## 🚀 快速开始

### 前置要求
- Node.js 18+
- npm 或 yarn
- PostgreSQL 数据库（或使用 Vercel Postgres）
- Git

### 5 分钟快速启动

```bash
# 1. 创建项目
npx create-next-app@latest xyvn-website --typescript --tailwind --app

# 2. 进入目录
cd xyvn-website

# 3. 安装依赖
npm install framer-motion @studio-freight/lenis @prisma/client resend

# 4. 初始化数据库
npx prisma init
npx prisma generate

# 5. 启动开发服务器
npm run dev
```

访问 http://localhost:3000

详细步骤请查看 **IMPLEMENTATION_GUIDE.md**
