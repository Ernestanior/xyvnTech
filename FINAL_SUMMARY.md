## 🚀 技术栈

### 核心技术
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **动画**: Framer Motion + GSAP + Lenis
- **3D**: Three.js + React Three Fiber
- **AI**: OpenAI GPT-4
- **数据库**: Prisma + PostgreSQL
- **部署**: Vercel

### 高级功能
- **音效**: Howler.js
- **手势**: @use-gesture/react
- **可视化**: D3.js + Chart.js
- **代码编辑**: Monaco Editor
- **虚拟滚动**: @tanstack/react-virtual
