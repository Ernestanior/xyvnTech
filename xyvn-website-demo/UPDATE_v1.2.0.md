## 📚 使用指南

### 查看新功能

1. **刷新浏览器**
   ```
   按 Ctrl + Shift + R（强制刷新）
   ```

2. **查看通知横幅**
   - 页面顶部会显示营销横幅
   - 可以点击"立即咨询"或关闭

3. **浏览新区块**
   - 数据统计：查看动画计数器
   - 工作流程：了解开发步骤
   - 价格方案：对比不同套餐
   - FAQ：点击问题查看答案

### 自定义内容

**修改数据统计**:
```tsx
// src/components/sections/StatsSection.tsx
const stats = [
  {
    value: 150,  // 修改数值
    suffix: '+', // 修改后缀
    label: '成功项目', // 修改标签
  },
];
```

**修改价格方案**:
```tsx
// src/components/sections/PricingSection.tsx
const plans = [
  {
    name: '基础版',
    price: '5万起', // 修改价格
    features: [...], // 修改功能列表
  },
];
```

**修改 FAQ**:
```tsx
// src/components/sections/FAQSection.tsx
const faqs = [
  {
    question: '你们的开发周期一般需要多久？',
    answer: '开发周期取决于...',
  },
];
```
